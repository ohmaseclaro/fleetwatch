Three providers, auto-discovered

Speaks the format every agent writes.

Each provider is independent and lazy — it surfaces session metadata immediately and only loads the heavy event stream when you open a session on your phone.

Claude Code

~/.claude/projects/*.jsonl

Live tailing via chokidar. Rotation-safe (inode + offset). Auto-discovers via env override, XDG path, or bounded filesystem search.

Cowork

Application Support / Claude / local-agent-mode-sessions

Same JSONL format, different parent path. Auto-discovered when Claude Cowork is installed; ignored cleanly when it isn't.

Cursor IDE

~/Library/Application Support/Cursor/User/globalStorage/state.vscdb

SQLite, read-only. WAL-file watching for live updates. Verifier checks for the cursorDiskKV table so VSCode's same-named DB never gets misidentified.

What's in the box

Sensible defaults, opinionated security.

Live WebSocket streaming

Events arrive on your phone within milliseconds of being written. No polling, no refresh.

Source tabs with brand icons

Claude, Cowork, Cursor — filter at a glance. Color-coded left-edge stripe for active sessions.

Image attachments + lightbox

Screenshots from screenshot tools and user-pasted images render inline. Tap to zoom.

Session info modal

Tap (i) on any session to see the transcript file path, project, git branch, size — with copy buttons.

Auto ngrok tunnel

Picks up ~/.../ngrok.yml automatically. Works on cellular, coffee shops, anywhere.

Optional password gate

Set PASSWORD=… in .env. Bcrypt-hashed in memory only; 30-day JWT after login.

Robust discovery

Env override → known paths → bounded filesystem search, with verify predicates so providers never collide.

Read-only by design

Never writes to source data. Opens DBs read-only. Never follows symlinks. Inodes tracked for safe rotation.

Install

One command. That's the whole setup.

No accounts. No config files. Sane defaults that work the moment you run it.

Run without installing

# Auto-discovers Claude / Cursor data, prints QR + URL.
npx @ohmaseclaro/fleetwatch

Install globally

npm install -g @ohmaseclaro/fleetwatch
fleetwatch

Enable ngrok (for access from anywhere)

# 1. Sign up free: https://dashboard.ngrok.com/signup
# 2. Copy your token: https://dashboard.ngrok.com/get-started/your-authtoken
# 3. Run with the token (persisted to ~/.config/fleetwatch/config.json):
fleetwatch --ngrok-authtoken <your-token>

Add a password (recommended with ngrok)

# bcrypt-hashed in memory only; never persisted to disk.
echo 'PASSWORD=correct horse battery staple' >> .env
fleetwatch
